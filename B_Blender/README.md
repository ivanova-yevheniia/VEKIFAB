# B_Blender — Work Package B: Blender Machine & Factory Generation

This work package generates **production machines and factory equipment**
procedurally in Blender using **Claude + the Blender MCP** integration.

It is one stage of the overall VEKIFAB pipeline:

```
Customer requirements
  → AI-generated production line plan
  → Factory description (machine-readable)
  → Blender Python generators (this work package)
  → .blend / .glb exports
  → Unity avatar walkthrough & inspection
```

## Goal of WP B

Given a machine-readable description of a production line, produce a **modular,
clean, Unity-ready 3D factory** in Blender. Each station is generated by a
dedicated, self-contained Python generator so the line can be assembled,
extended, or regenerated station-by-station.

## How it works

1. **Requirements in** — the customer's production requirements are captured in
   `requirements/customer_requirements.json`.
2. **Planning** — Claude turns those requirements into an ordered
   `production_line_plan.json` (which stations, in which order, with which
   parameters).
3. **Description** — the plan is expanded into a complete
   `factory_description.json` (positions, dimensions, connections, conveyors).
4. **Generation** — the Blender generators read the description and build the
   geometry. `generate_full_factory.py` orchestrates the individual station
   generators.
5. **Export** — the result is saved as `.blend` and exported to `.glb` for
   Unity, with preview screenshots.
6. **Handoff** — Unity imports the `.glb`; an avatar walks the line and
   inspects stations. See `docs/unity_import_notes.md`.

## Station generators

Each `generate_*.py` script is responsible for a single station type. They all
rely on `common_blender_utils.py` for consistent naming, materials, collision
setup, and placement so the exported scene stays clean and predictable for
Unity.

| Generator | Produces |
|-----------|----------|
| `generate_loading_station.py`   | Material / part loading station |
| `generate_assembly_station.py`  | Manual / semi-automatic assembly station |
| `generate_robot_cell.py`        | Robotic work cell (arm + safety fence) |
| `generate_vision_station.py`    | Camera / vision inspection station |
| `generate_functional_test.py`   | Functional / end-of-line test station |
| `generate_packaging_station.py` | Packaging & labeling station |
| `generate_storage_area.py`      | Storage / buffer / warehouse area |
| `generate_full_factory.py`      | Assembles all of the above into one line |
